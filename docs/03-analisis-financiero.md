# 1. Introducción

El presente documento analiza la estructura de costos, ingresos y proyecciones económicas del emprendimiento **AUMA**, con el fin de evaluar la viabilidad financiera del negocio y justificar la implementación del sistema e-commerce.

# 3. Costos Operativos Actuales

Actualmente, el negocio presenta una estructura de costos operativos reducida y manual:

### **Costos fijos actuales**
- Insumos de producción (variables, pero constantes en volumen)  
- Conectividad y mantenimiento de redes sociales  
- Transporte para entregas (variable)  
- Impresión de etiquetas  
- Participación en ferias (cuando aplica)  

### **Costos asociados al proceso manual**
- Tiempo de atención al cliente (mensajes, consultas, envío de fotos)  
- Tiempo de coordinación de pagos y entregas  
- Tiempo administrativo (registro manual de pedidos y stock)  

**Problema detectado:**  
El costo en tiempo operativo es elevado y no escalable.

# 7. Proyección con Implementación del Sistema E-commerce

La incorporación de un sistema e-commerce permitirá:

### **Ventajas económicas:**
- Incremento de ventas totales (acceso 24/7).  
- Mayor conversión gracias al carrito y catálogo organizado.  
- Reducción del tiempo manual dedicado a ventas.  
- Organización del inventario y pedidos.  
- Escalabilidad sin aumentar carga operativa.  
- Potencial para lanzar campañas pagas (meta ads) con landing profesional.  

### **Proyección estimada (ejemplo):**
- Aumento del 20–40% en ventas al primer año.  
- Ticket promedio más alto al presentar productos sugeridos.  
- Mayor retención gracias al login y seguimiento de pedidos.

# 8. Costos de Implementación del Sistema

### **8.1. Costos directos**
- Desarrollo inicial del MVP  
- Infraestructura AWS  
  - Amplify (mínimo en free tier)  
  - Cognito (pago según usuarios)  
  - DynamoDB (costos por lectura/escritura)  
  - API Gateway  
  - Lambda Functions  
  - CloudWatch  

> En fase MVP, los costos de AWS pueden ser muy bajos gracias al Free Tier.

### **8.2. Costos indirectos**
- Dominio web (opcional)  
- Diseño visual y branding del sitio  
- Eventuales integraciones de pago (comisiones por venta)

---

# 9. Análisis de Rentabilidad

La inversión en un e-commerce completo se recupera principalmente por:

- Aumento de ventas automatizadas  
- Reducción de horas de atención manual (que hoy son un costo)  
- Mejor organización del inventario  
- Disminución de errores en pedidos  
- Potencial para escalar sin contratar personal  

El proyecto tiene **alta viabilidad**, especialmente considerando el crecimiento orgánico actual de la marca y la demanda constante.

---

# 10. Conclusión Financiera

El análisis demuestra que AUMA tiene una base de costos reducida y un margen saludable que permite escalar el negocio mediante la digitalización.  
La inversión en el sistema e-commerce es financieramente justificable, ya que:

- optimiza procesos  
- aumenta ventas  
- reduce tiempos operativos  
- abre nuevos canales comerciales  
- profesionaliza la marca  

Este documento respalda el desarrollo del sistema propuesto y será utilizado como base para la planificación de requerimientos y fases posteriores del proyecto.
