# AUMA — Documentación del Proyecto

Este repositorio contiene la documentación oficial del proyecto AUMA.  
Centraliza la visión del negocio, el alcance del MVP, la planificación inicial y las decisiones estratégicas que guían su desarrollo.

AUMA es una marca de productos aromáticos y de bienestar elaborados de forma artesanal, con una identidad estética boho-natural y un enfoque en la calma, la energía suave y la experiencia sensorial del usuario.  
Este repositorio establece los fundamentos conceptuales y organizativos que estructuran su ecosistema digital.

## Contenido del repositorio


 ```
/docs
  └── vision.md     → Visión del negocio y alcance del MVP
  
```
El archivo `vision.md` incluye la visión del negocio y el alcance del MVP.  
En esta carpeta se incorporarán futuras ampliaciones como arquitectura técnica, decisiones de diseño, roadmap y documentación de sprints.

## Repositorios del proyecto AUMA

El proyecto se estructura en repositorios independientes para favorecer la escalabilidad, modularidad y buenas prácticas de desarrollo:

- auma-docs → documentación central del proyecto  
- auma-frontend → desarrollo de la interfaz web  
- auma-backend → API y lógica del negocio  

Cada repositorio cuenta con su propio README detallando su entorno, instalación y lineamientos técnicos.

## Objetivo de este repositorio

- Consolidar la documentación del proyecto AUMA como fuente única de referencia  
- Mantener la trazabilidad entre las decisiones de negocio, la planificación y la ejecución técnica  
- Establecer una base sólida para el desarrollo coordinado de frontend, backend e infraestructura  
- Facilitar futuras ampliaciones del proyecto bajo una estructura clara y escalable
