# Proveedores

> Documento oficial que define la gestión de proveedores de AUMA Studio.

---

# Objetivo

Documentar los criterios utilizados para seleccionar, clasificar y administrar los proveedores que abastecen a AUMA Studio.

Este documento constituye la referencia oficial para la gestión del abastecimiento de insumos utilizados por el emprendimiento.

---

# Alcance

Este documento incluye:

- clasificación de proveedores;
- criterios de selección;
- recursos abastecidos;
- criterios de evaluación;
- gestión de proveedores.

No incluye:

- precios de compra;
- costos;
- stock;
- procedimientos de producción;
- especificaciones técnicas de productos.

# 1. Clasificación de proveedores

Los proveedores se clasifican según el tipo de recurso que suministran al proceso productivo.

Las categorías actualmente utilizadas son:

- materias primas;
- envases;
- packaging;
- herramientas y equipamiento;
- servicios.

Cada proveedor podrá abastecer una o varias categorías.

---

# 2. Proveedores activos

AUMA Studio mantiene un listado de proveedores activos para abastecer los distintos recursos utilizados durante la producción.

Cada proveedor podrá suministrar una o más categorías de recursos.

La información deberá mantenerse actualizada cada vez que se incorpore un nuevo proveedor o se modifique el abastecimiento habitual.

---

# 3. Criterios de selección

La incorporación de un proveedor deberá evaluarse considerando, como mínimo:

- calidad de los productos;
- estabilidad del suministro;
- relación precio-calidad;
- disponibilidad de stock;
- tiempos de entrega;
- confiabilidad comercial.

La elección de un proveedor no deberá basarse únicamente en el precio.

---

# 4. Evaluación de proveedores

Los proveedores deberán revisarse periódicamente para verificar que continúan cumpliendo con los estándares definidos por AUMA Studio.

Podrán considerarse, entre otros, los siguientes aspectos:

- calidad constante;
- cumplimiento de entregas;
- disponibilidad de productos;
- atención comercial;
- evolución de precios.

---

# 5. Incorporación de nuevos proveedores

Todo nuevo proveedor deberá:

- abastecer un recurso necesario para la producción;
- cumplir los criterios de selección definidos por este documento;
- quedar registrado como proveedor activo cuando sea incorporado al proceso productivo.

La documentación de recursos y costos deberá actualizarse únicamente cuando la incorporación del proveedor modifique dichos procesos.
