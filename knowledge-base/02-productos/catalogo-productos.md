# Catálogo de Productos

> Documento que centraliza las líneas de productos de AUMA Studio y referencia la documentación específica de cada una.

---

# Objetivo

Centralizar las líneas de productos desarrolladas por AUMA Studio y servir como punto único de acceso a la documentación específica de cada una.

Este documento funciona como índice del dominio **Productos** y evita duplicar información que ya se encuentra documentada en las fichas individuales.

---

# Alcance

Este documento incluye:

* Líneas de productos existentes.
* Estado de desarrollo de cada línea.
* Referencias a su documentación específica.

No incluye:

* Especificaciones técnicas.
* Fragancias.
* Packaging.
* Información comercial.
* Información de producción.
