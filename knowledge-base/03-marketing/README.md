# Marketing

> Índice oficial de la documentación de Marketing de AUMA Studio.

---

# Objetivo

Centralizar la documentación relacionada con la comunicación, el posicionamiento y el crecimiento digital de AUMA Studio.

Este documento actúa como punto de entrada al dominio Marketing y referencia toda la documentación específica del área.

---

# Alcance

Este documento incluye:

- Estrategia general.
- Criterios editoriales.
- Comunicación de productos.
- Redes sociales.
- Biblioteca de contenidos.
- Campañas.

No incluye:

- Documentación de marca.
- Productos.
- Operaciones.
- Negocio.
